# SpearLeveling Plugin

A Minecraft 1.21+ plugin that adds leveling spears.

## Features
- Start with a **Wood Spear** (Unbreakable).
- Kill players to level up your spear.
- **Lunge Ability**: Right-click to dash forward (higher levels = longer dash).

## Tiers
1. **Wood**: Basic
2. **Stone**: Sharpness 1
3. **Copper**: Sharpness 2, Lunge 1
4. **Gold**: Sharpness 4, Lunge 1
5. **Iron**: Sharpness 5, Lunge 2
6. **Diamond**: Sharpness 7, Lunge 3
7. **Netherite**: Sharpness 10, Lunge 5

## Commands
- `/getspear`: Get the starting spear.

## Installation
1. Build the plugin using Maven.
2. Drop the jar into `plugins/`.
